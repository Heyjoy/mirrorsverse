## Design Context

See `.impeccable.md` for the canonical design context (users, brand personality, aesthetic direction, principles). All frontend/design work in this repo MUST respect that file.

**TL;DR:**
- Users: B2B 客户 / HR / 技术同行 / 中文聚会场景（4 类同页分层）
- Personality: 前卫 · 实验 · 锐利
- Visual: Dark first, 冷蓝紫 accent, 不对称布局, 大字号 display, 中英双语一等公民
- Anti: 通用 AI SaaS 模板 / 传统中式企业站 / 极简到无信息
- A11y: WCAG AA 底线
- 5 design principles: 信息密度 > 留白炫技 / 不对称胜对称 / 动效服务可读性 / 中文一等公民 / WCAG AA
